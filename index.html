Actions

Tap twice. Do anything.

Trigger a single shortcut, play back a sequence of keystrokes, or run any command on your Mac. Every trigger is yours to assign.

01 · Shortcut

Any keyboard shortcut

Any combination of plus a key. One shortcut per trigger — no conflicts, no modes to remember.

option ×2 → command + space
fn ×2 → command + shift + 4
02 · Chain

Multi-step sequences

One double-tap can play back up to three keystrokes in a row — each with its own modifiers. Perfect for repetitive actions like Select All → Copy → Switch window.

command ×2 →
command A → command C → command tab
03 · Shell

Run any command

Open websites, launch apps, run a Shortcut, or execute your own script. Runs quietly in the background so nothing skips a beat.

Menu bar, done right

Left-click toggles DoubleTap on and off. Right-click opens the menu. Behaves exactly like every other menu bar app on your Mac.

Liquid Glass throughout

Translucent sidebar, frosted cards, and system materials that match macOS 26 Tahoe.

Pure-Black Dark Mode

OLED-friendly solid black background, optional. Eyes thank you on external displays.

Launch at Login

Turn it on once and DoubleTap is ready the moment you sign in. No setup prompts, no background processes to manage.

Hide from Dock & Menu Bar

Polished visual effects

Privacy & safety

Your keyboard. Your Mac. Your business.

Runs entirely on your Mac. No tracking. Two macOS permissions, with status always visible in the app.

Signed, notarized, stapled

Every build is Developer ID signed, notarized by Apple, and stapled. No Gatekeeper workarounds.

Zero telemetry

No analytics, no tracking, no phone-home. Your mappings stay in your local Application Support folder.

Two permissions, granular

Accessibility and Input Monitoring — the standard macOS permissions any keyboard utility needs. If either is turned off, the menu bar tells you right away.

Automatic updates

Securely signed updates delivered straight from GitHub. Checks once a day in the background — or trigger it yourself from the menu.
